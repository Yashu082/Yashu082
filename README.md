## 🚀 Featured Projects

### 🩺 AI Health Bot + Medical NFTs on Aptos
> *Symptom classification + AI health tips + blockchain-secured health records*

A Streamlit app that classifies diseases from **770 classes** using a `RandomForestClassifier` trained on 378 symptoms. Accepts both **audio and text input**, integrates **Gemini AI** for personalized health recommendations, and mints health summaries as **NFTs on the Aptos blockchain (testnet)** — creating tamper-proof, portable medical records.

**Stack:** `Python` `Streamlit` `scikit-learn` `Gemini API` `Aptos Blockchain` `IPFS` `speech_recognition`

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Streamlit-FF4B4B?style=flat-square&logo=streamlit&logoColor=white)](https://hcns9f7q8mkk5sqqbsqzux.streamlit.app/)

### ⚙️ Predictive Maintenance Dashboard
> *Full-stack ML system for industrial equipment failure prediction*

End-to-end predictive maintenance system trained on **NASA's C-MAPSS turbofan dataset** (33,727 samples, 27 features). Outputs **failure probability (0–1)** with Low / Medium / High risk tiers. Served via a **Django REST API** and visualized through a **React + Plotly dashboard**.

**Stack:** `Python` `Django REST` `scikit-learn` `React` `Plotly.js` `Pandas` `NumPy`
**Model:** R² `0.87` · RMSE `0.059` · MAE `0.046`

[![Repo](https://img.shields.io/badge/GitHub-View%20Repo-181717?style=flat-square&logo=github)](https://github.com/Yashu082/predictive-maintenance-dashboard)

### 🚗 Road Accident Risk Prediction
> *ML model predicting accident risk from road & environmental factors*

Trains a `Random Forest Regressor` (300 trees) on **500,000+ road segments** across 20 features — road type, curvature, weather, lighting, and historical accident data. Deployed as an interactive **Streamlit app** with real-time prediction and CSV export.

**Stack:** `Python` `scikit-learn` `Pandas` `NumPy` `Streamlit`
**Model:** R² `0.8729` · RMSE `0.0592` · MAE `0.0460`

[![Repo](https://img.shields.io/badge/GitHub-View%20Repo-181717?style=flat-square&logo=github)](https://github.com/Yashu082/road-accident-risk-prediction)

### 📄 Resume Analyzer
> *Resume–job description matching with deterministic skill scoring + semantic similarity*

A Flask web app that extracts PDF resume text, applies **deterministic role–skill mapping** for explainable coverage scoring, and uses **sentence-transformers** (`stsb-roberta-large`) as a semantic similarity signal. Returns matched vs. missing skills with actionable suggestions.

**Stack:** `Python` `Flask` `PyMuPDF` `sentence-transformers` `Tailwind CSS` `pytest`

[![Repo](https://img.shields.io/badge/GitHub-View%20Repo-181717?style=flat-square&logo=github)](https://github.com/Yashu082)
